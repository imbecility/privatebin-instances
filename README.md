# privatebin-instances

получение свежих инстансов privatebin

"reliable" - содержит надежные инстансы и с высоким аптаймом

"low_uptime" - содержит достаточно стабильные инстансы но с аптаймом меньше 99%

"unreliable" - содержит инстансы, где данные могут быть удалены без предупреждения (не подходит для "вечного" хранения)
